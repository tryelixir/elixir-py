# Elixir

Python SDK for the Elixir API

## Installation

1. Clone the repository:

```bash
git clone https://github.com/acalabs-xyz/elixir-py
```

2. Navigate to the project directory:

```bash
cd elixir-py
```

3. Install the dependencies:

```bash
poetry install
```
